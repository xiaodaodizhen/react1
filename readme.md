## 安装
```
npm install create-react-app -g  安装手脚架

create-react-app project-filename  创建项目

- npm install axios 在react中发送ajax请求使用axios,封装了RESTFUL基于promise的，不支持jsonp

- npm install bootstrap@3.3.7

```
